Proj. 01

Atlas

Mobile strength & workout tracker. Offline-first. Own your data.

Shipped · production
Detail ▾

Design choices worth defending

  • Offline-first, on-device SQLite. No accounts, no backend, no data leaves the phone.
  • React Native + Expo for one-codebase iOS & Android without the Expo lock-in.
  • Body-part → exercise → set model. Optimised for how you actually train, not how databases store data.
  • Charts (weight, volume, PRs) deferred to v2. Deliberately shipped the smaller thing first.
Stack: React Native · Expo · TypeScript · SQLite · Reanimated 3

Proj. 03

Psychology Traits Shorts

Autonomous YouTube Shorts pipeline for cognitive-bias videos.

Autonomous · 28/28 tests
Detail ▾

Why it exists

  • Wanted to prove a full content pipeline could be autonomous after 2hr of setup.
  • Every step is tested. 28/28 green means adding a bias seeds a video without touching code.
  • Deliberate rate-limiting so the channel doesn’t look like a spam bot.
  • Stack chosen for portability. Whole thing runs on free GitHub Actions.
Stack: TypeScript · Node · FFmpeg · YouTube Data API · GH Actions · OpenAI
